# Finance Dashboard

## Features
-   **Financial Data Visualization**: Utilizes Recharts to generate visual reports and charts for better financial analysis.
-   **Revenue Prediction**: Implements a simple linear regression model to predict future revenue based on historical data.
-   **Dashboard View**: Presents a comprehensive dashboard with key performance indicators (KPIs), revenue trends, and other financial metrics.

## Installation

To run the application locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/Aareen098/Finance-Dashbaord.git
   cd Finance-Dashbaord
   ```

2.  Set up the client:
    *   Navigate to the `client` directory:
        ```bash
        cd client
        ```
    *   Install dependencies:
        ```bash
        npm install
        ```

## Usage

3.  Start the client:
    *   In the `client` directory, run:
        ```bash
        npm run dev
        ```

The application will be running at `http://localhost:5173`.
